## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
To have a copy of the repo and build something for yourself

1. Clone the repo
   ```sh
   git clone https://github.com/cyrusnderitu/job-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start Application
   ```sh
   npm start
   ```
